
Game name: The Dora Sisters

Team Name: Game Hack

Team Members: Aviral Kumar, Sheshansh Agrawal, Kanak Agrawal, Akash Trehan

Instructions to execute:  		[Linux- Ubuntu]

1. Install pygame. On ubuntu use the following command: sudo apt-get install python-pygame
2. Extract the folder gamehack.tar.gz. Make sure that image.jpg and music.ogg are in the same directory as that of the script.
3. Open terminal and run the script using: python file.py
4. The game rules are displayed during the game.
5. Play the game and enjoy :)
